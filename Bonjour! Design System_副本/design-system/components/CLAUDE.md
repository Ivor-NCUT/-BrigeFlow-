成员清单
Avatar.jsx: 用户头像组件，支持图片和文字 fallback
Badge.jsx: 徽标组件，支持数字和红点模式
Button.jsx: 按钮组件，支持 primary/secondary/ghost 变体
Card.jsx: 卡片容器，带阴影和圆角
CommentItem.jsx: 评论组件，支持递归嵌套回复
Grid.jsx: CSS Grid 布局容器
Input.jsx: 输入框组件，支持文本和图片粘贴/上传
ListItem.jsx: 列表项组件，用于消息列表等
NavBar.jsx: 顶部导航栏，支持左右操作区
TabBar.jsx: 底部标签栏，支持激活状态
Tag.jsx: 标签组件，支持多种颜色
index.js: 统一导出所有组件
